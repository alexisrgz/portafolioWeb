# Portafolio Web Personal

Este es mi portafolio web personal donde presento algunos de los proyectos que he desarrollado durante mi proceso de aprendizaje en el área de desarrollo de software.

## Sitio en Vivo

Puedes visitar mi portafolio aquí:

 https://alexisrgz.github.io/portafolioWeb/

## Acerca del Proyecto

Este portafolio fue diseñado para mostrar mis habilidades, proyectos y conocimientos en desarrollo web.  
El sitio tiene una interfaz moderna y responsive, enfocada en la experiencia del usuario y en la presentación visual de los proyectos.

El sitio incluye secciones como:

- Presentación / Sobre mí
- Tecnologías
- Proyectos desarrollados
- Información de contacto

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript

## Características

- Diseño responsive
- Interfaz interactiva
- Sección de proyectos
- Soporte para múltiples idiomas
- Diseño limpio y moderno
